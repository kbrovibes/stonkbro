# /project:list-backlog

Show current backlog status without modifying any files.

## Steps

1. Read BACKLOG.md and IDEAS.md.

2. Print a clean summary:

```
=== Backlog Status ===

🔄 In Progress: N items
  - [title]

✅ P1 — Do First: N items
  - [title]

📋 P2 — Do Next: N items
  - [title]
  ...

💡 IDEAS waiting for review: N items

✅ Done: N items total
```

3. Do not modify any files.
