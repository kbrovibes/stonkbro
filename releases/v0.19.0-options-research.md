# v0.19.0 — AI Options Research & CSP Picks

## Summary
Adds a dedicated "Options" tab and a deep-research task to find the top 10 cash-secured put (CSP) opportunities using the AI engine.

## Files changed
| File | Change |
|---|---|
| `src/components/BottomNav.tsx` | Added "Options" tab |
| `src/app/(app)/options/page.tsx` | New Options research dashboard |
| `src/lib/prompts/csp-picks.md` | Specialized AI prompt for top 10 CSPs |
| `src/lib/recommendations/generate.ts` | Backend logic for the new research theme |
| `src/app/api/recommendations/route.ts` | API support for specific theme fetching |
| `src/app/api/cron/options/route.ts` | New cron job for background refresh |

## Notes
- The "Top 10 CSP Picks" feature scans the market universe for stable uptrends and high-yield premiums.
- Data is cached in Supabase via the existing `daily_recommendations` table.
- A "Regenerate" button allows users to force a fresh AI scan if the current one is stale.
- The UI features a clean summary table followed by deep-dive technical reasoning for each pick.
