# v0.15.0 — Options Flow Scanner

**Date:** 2026-04-26

## What changed

Added an options flow analysis module that detects unusual activity patterns in options chains.

### New: `src/lib/options/flow-scanner.ts`

**Signal types detected:**

| Type | Detection Logic | Direction |
|------|----------------|-----------|
| `large_block` | Volume > 3x OI with 500+ contracts | Call = bullish, Put = bearish |
| `volume_wall` | Single strike > 15% of total OI (5000+ OI) | Call wall = resistance, Put wall = support |
| `otm_speculative` | > 10% OTM with 2000+ volume | Call = bullish, Put = bearish |
| `pcr_extreme` | PCR > 2.0 or < 0.3 | PCR > 2 = bearish, PCR < 0.3 = bullish |

**Significance scoring** (0-100): each signal scored by magnitude, top 5 averaged for composite `activityScore`.

**Sentiment**: classified bullish/bearish/neutral based on signal direction balance.

### New: `/api/flow`

**GET** `?tickers=NVDA,AAPL,...` — scans options chains for unusual activity:
- Default 15-ticker universe (high-options-volume names)
- Returns only tickers with detected signals
- Sorted by activity score descending
