# v0.12.0 — Scoring Engine

**Date:** 2026-04-26

## What changed

Populated the empty `src/lib/scoring/` directory with a quantitative stock scoring engine that rates stocks 0-100 across four dimensions.

### New files

- **`src/lib/scoring/types.ts`** — `ScoreBreakdown` and `ScoredStock` types
- **`src/lib/scoring/engine.ts`** — Pure scoring functions:
  - `scoreVolume(quote)` → 0-25 based on volume ratio thresholds (3x+ = max)
  - `scoreMomentum(quote)` → 0-25 based on price change magnitude, direction, volume confluence
  - `scoreTechnical(quote)` → 0-25 based on SMA position, 52-week range placement
  - `scoreEarnings(quote)` → 0-25 based on days until earnings (imminent = max catalyst)
  - `scoreStock(quote)` → composite `ScoredStock` with breakdown and letter grade
  - `scoreStocks(quotes)` → batch score + sort descending
- **`src/lib/scoring/index.ts`** — barrel export

### Updated files

- **`src/app/api/scanner/route.ts`** — replaced inline scoring with `scoreStocks()` from the engine

### Grading scale
