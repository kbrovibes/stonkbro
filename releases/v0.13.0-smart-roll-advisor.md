# v0.13.0 — Smart Roll Advisor

**Date:** 2026-04-26

## What changed

Added a multi-factor roll detection engine that evaluates short option legs against four roll triggers and produces unified, actionable recommendations.

### New: `src/lib/options/roll-advisor.ts`

**Four scoring factors** (each 0-30, capped at 100 total):

| Factor | Trigger | Max Score |
|--------|---------|-----------|
| DTE urgency | < 7 DTE = 25pts, < 3 = 28pts, expired = 30pts | 30 |
| Profit capture | > 50% of max = 20pts, > 80% = 30pts | 30 |
| Strike proximity | Stock within 2% of strike = 18pts, breached = 25-30pts | 30 |
| Earnings | < 5 days = 22pts, < 2 days = 30pts | 30 |

**Roll direction recommendations**:
- Short call breached → "Roll up and out — move strike above $X with 30-45 DTE"
- Short put breached → "Roll down and out — move strike below $X with 30-45 DTE"
- Low DTE → "Roll out — same strike, next monthly expiration"

**Urgency mapping**:
- Score >= 60 → high urgency (ROLL action)
- Score >= 35 → medium urgency (ROLL action)
- Score 20-34 → low urgency (WARNING action)

### Updated: `/api/signals`

- Smart roll alerts replace basic ROLL alerts for the same symbol+strike (no duplicates)
- Response now includes `rollRecommendations` array with full scoring breakdown
- All alerts still sorted by urgency

### Integration

The Today page's existing "Position Alerts" section already renders ROLL and WARNING alerts, so smart roll recommendations appear there automatically with enhanced detail messages.
