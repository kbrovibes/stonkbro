# v0.16.0 — Morning Briefing Email

**Date:** 2026-04-26

## What changed

Added a comprehensive morning briefing email that delivers a pre-market digest with everything needed to start the trading day.

### New: `src/lib/notifications/morning-briefing.ts`

Rich HTML email template with 5 sections:

| Section | Content | Priority |
|---------|---------|----------|
| Action Required | High-urgency alerts (roll, close, stop triggered) | Red |
| Expiring This Week | Short options with DTE <= 7 | Red |
| Earnings Today | Tickers reporting today/tomorrow with BMO/AMC timing | Amber |
| Market Pulse | Top 8 explosive movers with change % and volume | Purple |
| AI Picks | Latest 5 recommendations from daily engine | Blue |

Subject lines adapt to content urgency.

### New: `/api/cron/morning-briefing`

Cron endpoint that:
1. Gathers movers + earnings data in parallel
2. Per user: fetches active positions, generates alerts + smart roll recommendations, finds expiring legs
3. Pulls latest AI recommendations from Supabase
4. Sends personalized morning briefing to each user with alert_email configured

**Recommended Vercel cron schedule:** `0 13 * * 1-5` (9:00 AM ET, weekdays)
