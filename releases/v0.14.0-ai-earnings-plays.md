# v0.14.0 — AI Earnings Plays

**Date:** 2026-04-26

## What changed

Added a pre-earnings strategy generator that analyzes options chains for upcoming earnings and uses AI to suggest specific plays.

### New: `src/lib/options/earnings-plays.ts`

- `analyzeEarningsIV(quote, chain, event)` — analyzes options chain for earnings context:
  - Finds post-earnings expiration
  - Calculates ATM straddle cost and expected move %
  - Computes average IV across near-term chain
  - Returns `EarningsPlayData` with full IV analysis

- `buildEarningsPlayPrompt(plays)` — constructs AI prompt requesting specific strategies:
  - CSP, iron condor, credit spreads, straddle/strangle
  - Considers expected move, IV level, timing (BMO/AMC)
  - Returns JSON array of strategy suggestions with strikes, premium, max risk/reward

### New: `/api/earnings-plays`

**GET** — scans 24-ticker universe for earnings in next 14 days:
1. Fetches earnings calendar
2. Gets live quotes for upcoming tickers
3. Fetches options chains and analyzes IV (sequential for rate limits)
4. Generates AI strategy suggestions for tickers with chain data

**Response:**
```json
{
  "plays": [...],          // EarningsPlayData per ticker
  "suggestions": [...],    // AI strategy recommendations
  "earningsCount": 5,
  "chainsAnalyzed": 5
}
```
